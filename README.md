# Income Tax Calculator

A simple, interactive income tax calculator that provides detailed tax breakdowns for different filing statuses. Built with vanilla HTML, CSS, and JavaScript for easy calculation of federal income taxes.

**🔗 [Live Demo](https://btaira.github.io/IncomeTaxCalculator/tax-breakdown.html)**

## Features

- **Interactive Income Input**: Enter your income via text input or slider
- **Real-time Calculations**: Instant tax calculations as you adjust income
- **Multiple Filing Statuses**: Support for different tax filing categories
- **Detailed Tax Breakdown**: Visual representation of tax liability and effective tax rates
- **Dark Theme**: Modern, easy-on-the-eyes interface
- **Responsive Design**: Works on desktop and mobile devices
- **No Dependencies**: Pure HTML, CSS, and JavaScript—no external libraries required

## Overview

This calculator helps you understand your federal income tax obligations by showing:
- Gross income
- Taxable income
- Tax liability for your filing status
- Effective tax rate
- Visual breakdown of tax calculations

## Usage

### Online
Visit the [live demo](https://btaira.github.io/IncomeTaxCalculator/tax-breakdown.html) to use the calculator directly in your browser.

### Local
1. Clone the repository:
   ```bash
   git clone https://github.com/btaira/IncomeTaxCalculator.git
   cd IncomeTaxCalculator
   ```
2. Open `tax-breakdown.html` in your web browser
3. Enter your income using the input field or slider
4. Select your filing status (Single, Married, Head of Household, etc.)
5. View your tax breakdown instantly

## How It Works

The calculator:
1. Takes your gross income as input
2. Applies applicable deductions and exemptions based on filing status
3. Calculates federal income tax using current tax brackets
4. Displays comprehensive breakdown of your tax liability
